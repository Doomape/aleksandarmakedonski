== Description ==

A plugin to replace the default 'gallery' shortcode and add additional HTML tags for more customization. Displaying Caption, Title and Description of the image if those variables are set for images.

Display:
*   Photo/Image Caption
*   Photo/Image Title
*   Photo/Image Description

[See plugin in action](http://ronimarinkovic.com/photo-exhibition-2010/ "See plugin in action")

[More about Enhanced WP Gallery](http://ronimarinkovic.com/wordpress/enhanced-wordpress-gallery-shortcode-plugin/ "More about Enhanced WP Gallery")

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload  `EnhancedWPGalery`  folder to the  `/wp-content/plugins/`  directory
2. Activate the plugin through the 'Plugins' menu in WordPress
